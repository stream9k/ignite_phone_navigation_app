# 차량용 내비게이션 자동화 앱 (Ignite)

이 앱은 차량에 고정된 안드로이드 폰을 내비게이션 전용으로 사용할 때, 전원 연결 상태에 따라 앱 실행과 종료를 자동화하여 운전자의 편의를 돕기 위해 개발되었습니다.

## 1. 주요 기능

- **내비게이션 자동 실행**: 차량 시동을 걸어 스마트폰에 전원이 연결되면, Tmap 내비게이션을 자동으로 실행합니다.
- **앱 자동 종료**: 차량 시동을 끄면(전원 해제), 설정한 시간(초 단위) 후에 실행 중인 모든 앱을 종료하여 배터리를 절약합니다.
- **시스템 자동 종료**: 차량 시동을 끄고 설정한 시간(분 단위)이 지나면 휴대폰 전원을 완전히 끕니다. (장기 주차 시 배터리 방전 방지)
- **기능 일시 정지 (마스터 스위치)**: 휴대폰을 차에서 분리하여 평소처럼 사용할 때, 자동 실행/종료 기능이 작동하지 않도록 간편하게 끄고 켤 수 있습니다.

## 2. 초기 설정 가이드 (필수)

앱을 처음 설치한 후 다음 순서대로 설정해 주세요.

### 가. 기능 설정
1.  **자동 실행 및 종료 기능 사용**: 스위치를 **ON**으로 둡니다. (휴대폰을 차에서 떼어낼 땐 **OFF**로 변경)
2.  **앱 자동 종료 대기 시간**: 전원이 꺼진 후 내비게이션을 종료할 시간을 **초 단위**로 입력하고 [저장]을 누릅니다. (기본값: 60초)
3.  **휴대폰 전원 끄기 대기 시간**: 전원이 꺼진 후 휴대폰 자체를 종료할 시간을 **분 단위**로 입력하고 [저장]을 누릅니다. (기본값: 90분)

### 나. 권한 설정 (순서대로 진행)
버튼을 차례대로 눌러 권한을 허용해 주세요.

1.  **알림 권한 요청**: 앱의 상태 알림을 표시하기 위해 허용합니다.
2.  **전체 화면 알림 설정**: 화면이 꺼져 있을 때도 내비게이션을 실행하기 위해 허용합니다.
3.  **다른 앱 위에 표시 허용**: 다른 앱(Tmap)을 실행하기 위해 권한을 허용합니다.
4.  **접근성 설정**: 
    *   **Android 13 이상 사용자**: [4-1. 제한된 설정 허용] 버튼을 먼저 눌러 제한을 푼 뒤, [4. 접근성 설정]을 눌러 `Ignite` 서비스를 켜주세요.
    *   **그 외 사용자**: [4. 접근성 설정] 버튼을 눌러 `Ignite` 서비스를 켜주세요.
5.  **배터리 최적화 해제**: 백그라운드에서 앱이 죽지 않도록 최적화를 해제합니다.

## 3. 수동 테스트
설정이 잘 되었는지 확인하기 위해 아래 버튼을 눌러보세요.
- **Tmap**: Tmap이 즉시 실행되는지 확인합니다.
- **앱 종료**: 실행 중인 모든 앱이 종료되고 홈 화면으로 돌아가는지 확인합니다.
- **시스템 종료**: 휴대폰 전원 끄기 팝업이 뜨고 자동으로 종료되는지 확인합니다.

---
**주의사항**: 삼성 기기 등 일부 제조사 폰에서는 '배터리 최적화 해제' 설정이 풀릴 수 있으니, 앱이 동작하지 않을 때 다시 확인해 주세요.

# 차량용 내비게이션 자동화 앱 (Ignite)

## 1. 앱의 목적

이 앱은 차량에 거치된 안드로이드 폰을 자동으로 관리하여 운전자의 편의를 돕기 위해 개발되었습니다.

- **내비게이션 자동 실행**: 차량 시동을 걸어 스마트폰에 전원이 연결되면, 지정된 내비게이션 앱(Tmap)을 자동으로 실행합니다.
- **모든 앱 자동 종료**: 차량 시동을 끄면(전원 해제), 사용자가 설정한 시간(분 단위) 후에 실행 중인 모든 앱을 종료하여 배터리 소모를 방지합니다.


## 2. 앱 사용 방법 (필수 설정)

앱이 정상적으로 동작하려면 초기 설정이 반드시 필요합니다.

### 가. 자동 종료 시간 설정

1.  `Ignite` 앱을 엽니다.
2.  '자동 종료 시간 설정' 아래의 입력창에 원하는 시간을 **분 단위**로 입력합니다.
3.  **[저장]** 버튼을 눌러 설정을 완료합니다.

### 나. 접근성 서비스 권한 허용

이 권한은 다른 앱을 종료시키거나 화면을 잠그는 핵심 기능에 반드시 필요합니다.

1.  `Ignite` 앱의 **[1. 접근성 설정 열기]** 버튼을 누릅니다.
2.  설정 화면에서 **[설치된 서비스]** 또는 **[다운로드한 앱]** 메뉴로 들어갑니다.
3.  목록에서 **`Ignite`**를 찾아 스위치를 **[사용 중]**으로 변경합니다.

### 다. 배터리 최적화 해제

안드로이드 시스템이 앱을 강제로 절전 상태로 만들지 않도록 하여, 백그라운드에서 안정적으로 동작하게 합니다.

1.  `Ignite` 앱의 **[2. 배터리 최적화 해제]** 버튼을 누릅니다.
2.  (만약 화면이 열리지 않으면) 폰의 **설정** 앱을 열고 **'배터리 사용량 최적화'**라고 검색합니다.
3.  앱 목록 상단의 드롭다운 메뉴를 **[전체]**로 변경합니다.
4.  목록에서 **`Ignite`**를 찾아 스위치를 **해제**합니다.

--- 

모든 설정이 완료되었습니다. 이제 차량에서 충전기를 연결하거나 해제하면 앱이 자동으로 동작합니다.

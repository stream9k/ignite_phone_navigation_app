# 차량용 안드로이드 자동화 앱 (Ignite)

차량에 고정된 안드로이드 기기를 사용할 때, 차량의 전원(시동) 상태에 따라 지정된 앱을 자동으로 실행하고, 전원이 꺼지면 앱 종료 후 시스템 종료 또는 비행기 모드로 전환하여 기기 관리를 자동화합니다.

## 1. 주요 기능

- **다중 앱 자동 실행**: 차량 시동을 걸어 스마트폰에 전원이 연결되면, 지정된 앱들을 순서대로 자동 실행합니다.
  - `+`, `-` 버튼을 통해 여러 개의 앱을 목록에 추가하거나 삭제할 수 있습니다.
- **실행 앱 자동 종료**: 차량 시동을 끄면(전원 해제), 설정한 시간(초 단위) 후에 실행 중인 모든 앱을 종료하여 배터리를 절약합니다.
- **최종 동작 자동화**: 앱 종료 후, 설정된 대기 시간(분 단위)이 지나면 다음 3가지 동작 중 하나를 수행합니다.
  1.  **시스템 종료**: 기기 전원을 완전히 끕니다. (장기 주차 시 배터리 방전 방지)
  2.  **비행기 모드**: 네트워크를 차단하여 대기 전력을 최소화합니다. (다음에 시동 걸면 비행기 모드 자동 해제)
  3.  **없음**: 앱 종료 후 아무 동작도 하지 않고 화면만 꺼진 상태로 대기합니다.
- **기능 일시 정지**: 메인 스위치를 통해 모든 자동화 기능을 간편하게 끄고 켤 수 있습니다.

## 2. 초기 설정 가이드 (필수)

앱의 모든 기능을 사용하려면 아래 권한들을 순서대로 설정해야 합니다.

1.  **알림 권한 요청**: 앱의 상태 알림을 표시하기 위해 필요합니다.
2.  **전체 화면 알림 설정**: 화면이 꺼져 있을 때도 앱을 실행하기 위해 필요합니다.
3.  **다른 앱 위에 표시 허용**: 다른 앱을 화면 위에 띄우기 위해 필요합니다.
4.  **접근성 설정**: 
    *   **Android 13 이상**: [4-1. 제한된 설정 허용] 버튼을 먼저 눌러 앱 정보 화면으로 이동 후, 우측 상단 메뉴(⋮)에서 '제한된 설정 허용'을 눌러야 합니다. 그 다음, [4. 접근성 설정] 버튼을 눌러 `Ignite` 서비스를 켤 수 있습니다.
    *   **그 외 버전**: [4. 접근성 설정] 버튼을 눌러 `Ignite` 서비스를 켜주세요.
    *   *이 권한은 '모든 앱 종료', '시스템 종료', '비행기 모드' 등 화면을 자동으로 클릭하는 핵심 기능에 필수적입니다.*
5.  **배터리 최적화 해제**: 앱이 백그라운드에서 시스템에 의해 종료되지 않도록 설정합니다.

## 3. 사용 방법

### 가. 기능 활성화
- **Ignite 앱 기능 전체 사용**: 이 스위치를 **ON**으로 설정해야 모든 자동화 기능이 동작합니다. (기본값: ON)

### 나. 자동 실행 앱 설정
1.  `[자동시작할 내비게이션 앱 선택]` 목록에서 앱 선택 버튼을 누릅니다.
2.  표시되는 앱 목록에서 원하는 앱을 검색하여 선택합니다.
3.  `[+]` 버튼을 눌러 실행할 앱을 추가할 수 있습니다.
4.  `[-]` 버튼(두 번째 줄부터 표시)을 눌러 해당 앱을 목록에서 삭제할 수 있습니다.

### 다. 지연 시간 및 최종 동작 설정
1.  **앱 종료 대기**: 시동이 꺼진 후, 실행 중인 모든 앱이 종료될 때까지의 시간을 초 단위로 입력하고 [저장]합니다.
2.  **최종 동작 설정**: `시스템 종료`, `비행기 모드`, `없음` 중 원하는 동작을 선택합니다.
3.  **동작 대기**: '없음'을 선택하지 않은 경우, 앱 종료 후 최종 동작이 실행될 때까지의 시간을 분 단위로 입력하고 [저장]합니다.

### 라. 수동 테스트
설정이 올바르게 되었는지 아래 버튼으로 즉시 테스트할 수 있습니다.
- **내비 실행**: 설정된 앱 목록을 순서대로 실행합니다.
- **앱 종료**: '모두 닫기' 기능을 수행합니다.
- **시스템 종료**: 시스템 종료를 수행합니다.
- **비행기 모드 토글 테스트**: 비행기 모드를 켜거나 끕니다.

---
**주의사항**: 이 앱은 접근성 서비스를 이용하여 화면의 버튼(예: 모두 닫기, 전원 끄기)을 자동으로 클릭합니다. 기기의 OS 버전이나 제조사 UI에 따라 버튼의 이름이나 위치가 달라 정상적으로 작동하지 않을 수 있습니다.
